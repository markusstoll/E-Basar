# Contributor Code of Conduct

## Our pledge

We as contributors and maintainers pledge to make participation in this project a harassment-free experience for everyone. We expect respectful, constructive communication when opening issues, discussing ideas, or submitting pull requests.

## Expected behaviour

- Be respectful and inclusive.
- Accept constructive criticism and focus on what is best for the project.
- Keep discussions focused on the project and its use cases.

## Unacceptable behaviour

- Personal attacks, trolling, or insulting comments.
- Public or private harassment.
- Other conduct that could reasonably be considered inappropriate in a professional or open source setting.

## Scope

This code of conduct applies in project spaces (e.g. GitHub issues and pull requests) and in public spaces when representing the project.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, or other contributions that do not align with this code of conduct. Repeated or serious violations may result in being blocked from the repository.

## Attribution

This code of conduct is adapted from common open source community standards and is not legally binding. By participating, you are expected to uphold these guidelines.
