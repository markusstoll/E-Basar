# Roadmap

Planned features for upcoming releases. Contributions and feedback are welcome via [GitHub Issues](https://github.com/markusstoll/E-Basar/issues).

## Planned features

### Multilingualism (i18n)

- **Goal**: Support multiple languages for the UI (e.g. German, English).
- **Scope**: Localizable strings, language selector, persisted language preference.
- **Status**: in work.

### PayPal payment

- **Goal**: Allow buyers to pay via PayPal in addition to (or instead of) real-time bank transfer and WERO.
- **Scope**: PayPal integration for buyer payments; compatibility with existing balancing and protocol.
- **Status**: Planned.

### WERO payment for seller payouts

- **Goal**: Support WERO not only for buyer payments but also for paying out sellers (in addition to real-time transfer).
- **Scope**: WERO-based payout flow for sellers; optional choice between Echtzeitüberweisung and WERO per payout.
- **Status**: Planned.
